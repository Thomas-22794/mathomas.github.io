📈 Suites numériques

1. Suite arithmétique

Une suite (uₙ) est arithmétique s'il existe une raison r telle que :

Terme général
uₙ = u₀ + n·r ou uₙ = u_p + (n−p)·r
Somme des termes
S = (nombre de termes) × (premier + dernier) / 2
La raison r = uₙ₊₁ − uₙ est CONSTANTE. Si r > 0 → croissante ; r < 0 → décroissante.

2. Suite géométrique

Une suite (uₙ) est géométrique s'il existe une raison q ≠ 0 telle que :

Terme général
uₙ = u₀ × qⁿ ou uₙ = u_p × qⁿ⁻ᵖ
Somme des termes (q ≠ 1)
S = u₀ × (1 − qⁿ⁺¹) / (1 − q) = u₀ × (qⁿ⁺¹ − 1) / (q − 1)
Si |q| < 1 → la suite tend vers 0. Si q > 1 → tend vers +∞. Si q = 1 → constante.

3. Suites récurrentes uₙ₊₁ = f(uₙ)

Pour étudier la convergence :

  • Chercher les points fixes : résoudre f(x) = x
  • Étudier la monotonie pour montrer que la suite est bornée et monotone
  • Si convergente vers ℓ alors ℓ est un point fixe : ℓ = f(ℓ)
  • Utiliser une suite auxiliaire (vₙ = uₙ − ℓ) pour se ramener à une suite géométrique
Pour une suite du type uₙ₊₁ = a·uₙ + b, la limite est le point fixe ℓ = b/(1−a), et vₙ = uₙ − ℓ est géométrique de raison a.

4. Limites de suites classiques

À connaître par cœur
lim qⁿ = 0 si |q| < 1 lim qⁿ = +∞ si q > 1 lim n^α = +∞ pour tout α > 0 lim (1 + 1/n)ⁿ = e (nombre d'Euler)
Croissances comparées : ln(n) ≪ n^α ≪ qⁿ ≪ n! quand n → +∞

📐 Fonction logarithme & exponentielle

1. Fonction exponentielle

Propriétés fondamentales
exp(a+b) = exp(a)·exp(b) exp(a−b) = exp(a)/exp(b) exp(n·a) = [exp(a)]ⁿ (eˣ)' = eˣ (e^(f(x)))' = f'(x)·e^(f(x))
eˣ > 0 pour tout x ∈ ℝ. La fonction exp est strictement croissante sur ℝ.

2. Fonction logarithme népérien

Propriétés
ln(a·b) = ln(a) + ln(b) ln(a/b) = ln(a) − ln(b) ln(aⁿ) = n·ln(a) ln(1) = 0 , ln(e) = 1 (ln x)' = 1/x sur ]0,+∞[ (ln(f(x)))' = f'(x)/f(x)
ln et exp sont réciproques : ln(eˣ) = x et e^(ln x) = x pour x > 0.

3. Croissances comparées (CRUCIAL pour le bac)

Quand x → +∞
lim (ln x)/xⁿ = 0 pour tout n > 0 lim xⁿ/eˣ = 0 pour tout n > 0 lim eˣ = +∞ et lim e^(−x) = 0
Quand x → 0⁺
lim x·ln(x) = 0 lim ln(x) = −∞

4. Convexité

  • f est convexe sur I si f'' ≥ 0 sur I (courbe au-dessus des tangentes)
  • f est concave sur I si f'' ≤ 0 sur I (courbe en-dessous des tangentes)
  • Point d'inflexion : f'' change de signe
eˣ est convexe sur ℝ (exp'' = eˣ > 0). ln est concave sur ]0,+∞[ (ln'' = −1/x² < 0).

5. Équations différentielles

y' = ay → solutions
y = C·eᵃˣ (C ∈ ℝ quelconque)
y' = ay + b → solutions
y = (y₀ + b/a)eᵃˣ − b/a (solution particulière yₚ = −b/a)

∞ Limites de fonctions

1. Formes indéterminées (FI)

Formes indéterminées à lever
∞ − ∞ | 0 × ∞ | ∞/∞ | 0/0 | 1^∞ | 0⁰ | ∞⁰
Pour ∞/∞ avec polynômes : factoriser par le terme de plus haut degré. Pour 0/0 : factoriser ou utiliser la dérivée (règle de l'Hospital).

2. Opérations sur les limites

Règles
lim[f+g] = lim f + lim g (sauf FI ∞−∞) lim[f×g] = lim f × lim g (sauf FI 0×∞) lim[f/g] = lim f / lim g (sauf FI 0/0 ou ∞/∞)
+∞ + (−∞) est une forme INDÉTERMINÉE. Il faut toujours lever les FI !

3. Théorème des gendarmes

Si g(x) ≤ f(x) ≤ h(x) au voisinage de a et lim g = lim h = L, alors lim f = L.

Très utile pour les fonctions oscillantes (sin, cos) : −1 ≤ sin(x) ≤ 1 donc |sin(x)/x| ≤ 1/|x| → 0.

4. Continuité et TVI

  • f continue sur [a,b] et f(a) et f(b) sont de signes opposés → f s'annule au moins une fois sur ]a,b[
  • Si de plus f est strictement monotone → il existe un UNIQUE c ∈ ]a,b[ tel que f(c) = 0
TVI = Théorème des Valeurs Intermédiaires. Toujours vérifier : continuité + valeurs encadrant 0.

5. Limites usuelles à connaître

Au voisinage de 0
lim(x→0) sin(x)/x = 1 lim(x→0) (eˣ−1)/x = 1 lim(x→0) ln(1+x)/x = 1

2. Calcul d'intégrale

Définition
∫[a,b] f(x)dx = [F(x)]ᵃᵇ = F(b) − F(a)
L'intégrale représente l'aire algébrique entre la courbe et l'axe des abscisses.
Propriétés
∫[a,b] [f+g] = ∫f + ∫g ∫[a,b] λ·f = λ·∫[a,b] f ∫[a,b] f = −∫[b,a] f ∫[a,a] f = 0

3. Intégration par parties (IPP)

Formule IPP
∫[a,b] u·v' = [u·v]ᵃᵇ − ∫[a,b] u'·v
Choix de u et v' : u doit se simplifier en dérivant, v' doit être facile à intégrer. Ordre LIATE : Logarithme, Inverse, Algébrique, Trigono, Exponentielle → u est l'élément le plus à gauche.

4. Valeur moyenne

🎲 Probabilités

1. Loi binomiale B(n, p)

X suit une loi binomiale B(n,p) si X = nombre de succès en n épreuves indépendantes de Bernoulli de probabilité p.

Probabilité
P(X = k) = C(n,k) × pᵏ × (1−p)^(n−k)
Espérance et variance
E(X) = n·p V(X) = n·p·(1−p) σ(X) = √(n·p·(1−p))
C(n,k) = n! / (k!(n−k)!) — coefficient binomial "k parmi n".

2. Loi normale N(μ, σ²)

Propriétés
E(X) = μ , σ(X) = σ P(μ−σ ≤ X ≤ μ+σ) ≈ 0.683 P(μ−2σ ≤ X ≤ μ+2σ) ≈ 0.954 P(μ−3σ ≤ X ≤ μ+3σ) ≈ 0.997
Loi normale centrée réduite N(0,1) : X standardisé → Z = (X−μ)/σ. On lit les valeurs dans la table.

3. Approximation de la loi binomiale

  • Si n ≥ 30, np ≥ 5 et n(1−p) ≥ 5 → approximation par N(np, np(1−p))
  • Si n grand et p petit (np < 5) → approximation par la loi de Poisson

4. Intervalle de fluctuation

Au seuil 95% pour fréquence f = X/n
I_f = [p − 1/√n , p + 1/√n]
Si la fréquence observée est hors de l'intervalle → on remet en cause l'hypothèse sur p.

5. Espérance d'une variable aléatoire discrète

Définition
E(X) = Σ xᵢ · P(X = xᵢ) V(X) = E(X²) − [E(X)]²
Exercice 1 — Espérance binomiale

📦 Géométrie dans l'espace

1. Vecteurs de l'espace

Coordonnées — opérations
u⃗ = (x,y,z) , v⃗ = (x',y',z') u⃗ + v⃗ = (x+x', y+y', z+z') λu⃗ = (λx, λy, λz) ‖u⃗‖ = √(x²+y²+z²)

2. Produit scalaire dans l'espace

Formules
u⃗·v⃗ = xx' + yy' + zz' u⃗·v⃗ = ‖u⃗‖·‖v⃗‖·cos(θ) u⃗ ⊥ v⃗ ⟺ u⃗·v⃗ = 0

3. Équation d'un plan

Plan de vecteur normal n⃗=(a,b,c) passant par A(x₀,y₀,z₀)
ax + by + cz + d = 0 avec d = −(ax₀ + by₀ + cz₀)
Le vecteur normal au plan ax+by+cz+d=0 est n⃗ = (a,b,c). Deux plans sont parallèles si leurs normales sont colinéaires.

4. Équation d'une droite

Représentation paramétrique — droite passant par A(x₀,y₀,z₀) de vecteur directeur u⃗=(l,m,n)
x = x₀ + tl y = y₀ + tm z = z₀ + tn (t ∈ ℝ)

5. Distance d'un point à un plan

Distance de M(x₀,y₀,z₀) au plan ax+by+cz+d=0
d = |ax₀+by₀+cz₀+d| / √(a²+b²+c²)

6. Positions relatives

  • Droite ∥ plan : vecteur directeur ⊥ normale du plan (produit scalaire = 0) ET point de la droite ∉ plan
  • Droite ⊂ plan : vecteur directeur ⊥ normale ET point ∈ plan
  • Droite ∩ plan : substituer la paramétrique dans l'équation du plan
Toujours vérifier avec UN point si la droite est dans le plan ou seulement parallèle !
